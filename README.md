# 🚨 LAPOR BENCANA

Website pelaporan bencana berbasis web yang mengirimkan laporan langsung ke Telegram.

**➡️ [Buka Aplikasi](https://kingsyah.github.io/lapor-bencana/)**

## Fitur

- 📍 Peta interaktif + GPS otomatis
- 📷 Lampiran foto/video
- 💬 Kirim langsung ke Telegram
- 📱 Bisa diinstal ke homescreen (PWA)
- 🔒 Anti-spam & aman

## Tech Stack

HTML · CSS · JavaScript · Leaflet.js · Supabase · Telegram Bot API
